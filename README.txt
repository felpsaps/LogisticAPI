LogisticAPI

Pre-requisitos.
JDK 1.8
Tomcat 7

Intruçoes de Build e deploy:
Link do projeto no GitHub: https://github.com/felpsaps/LogisticAPI.git
 - Clonar o projeto
 - Criar o projeto no Eclipse
 - Executar a build do maven no arquivo pom.xml para baixar as dependencias
 - Executar o comando clean install do mavem para gerar um arquivo .war
 - Para o deploy, colocar o war dentro da pasta webapps do tomcat
 - Caso prefira, é possivel rodar direto pelo Eclipse
